🚀 TaxChain — Blockchain-Based Tax Management System

TaxChain is a decentralized application (dApp) built on the Solana blockchain that enables secure, transparent, and tamper-proof tax record management. It allows users to submit, store, and view tax records using smart contracts.

📌 Features
🔐 Secure tax record storage on blockchain
📊 Transparent and immutable transaction history
👤 Wallet-based user interaction (Phantom, Solflare)
⚡ Fast and low-cost transactions using Solana
💻 Modern UI dashboard for managing records
